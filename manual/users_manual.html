USER'S MANUAL FOR RESERVATION SYSTEM

Admin Log In

  • To access the information, logging in the admin account is required.
  • To log in, fill up the required field and click the OK button.

  • After correctly logging in, a message will pop up.
  • Then click OK.
  • The main form or the dashboard will open.

Reservation Dashboard / Forms

  • Choose and click any button you want to open. In this case, the Customer Information is chosen.

Customer Information

  • To insert, update and delete customer information, just fill

up the 4 necessary information and click the button.

  • To view the customer information, just click the load button.

  • To print and export the data in the datagrid in order to generate a report, click the print button.

  • After clicking the print button, a message (name of the report) will pop up and click OK.
  • Then, the report will automatically open.

  • To access other information for reservation, just click the Back to Main Form button to go back to the dashboard.

  • This manual in Customer information is the same as Reservation, Event, Venue and Venue Profit Information.

  • For Data Loading form/ information, click the Show Data Loading Form from the dashboard form.

  • To upload csv file, click the select csv button. Then, select the csv file that you want to open and upload.
  • After selecting, the data in the csv file will automatically display in the datagrid.

  • To create a backup database, just click the Backup
  • Database button.

  • The folder dialog will open and select then open the batch file.

  • After selecting the batch file, the folder dialog will also open for saving the backup file.

  • Lastly, the message will pop up that tha backup database is successfully created. Then click OK button.

  • To exit the data loading form, just click the Back to Main Form button.

  • From the Main form / Dashboard, click the Exit Button to close the program.
